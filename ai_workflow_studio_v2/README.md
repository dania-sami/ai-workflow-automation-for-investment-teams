# AI Workflow Studio

This project explores a multi-agent approach to workflow intelligence and optimization.

## Features

- Multi-agent system (Research, Ops, Risk, Strategy, Reporting)
- Workflow scoring engine
- SOP+ generator
- Insights dashboard
- AI assistant chat interface
- Modern UI with gradient design

## Why I built this

I wanted to explore how AI can evolve from single tools into full workflow systems.

## Tech

- Python
- Streamlit
- Pandas

## Run

pip install -r requirements.txt
streamlit run app.py
